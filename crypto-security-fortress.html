🔐 Layer 1: Hardware Wallet Mandate

Rule: Any crypto holding >$1,000 MUST be on a hardware wallet.

Recommended: Ledger Nano S Plus ($79) or Trezor Model One ($69)
Buy ONLY from ledger.com or trezor.io. NEVER Amazon/eBay.
  • Set up on a CLEAN computer (no malware)
  • Write seed phrase on PAPER with a PEN (not pencil, not digital)
  • Store in 2+ physical locations (home safe + bank safe deposit)
  • NEVER screenshot, photograph, or type seed phrase on any device

✅ Layer 4: Transaction Verification Protocol

The 5-Second Rule (Before EVERY transaction):

  • 1. Verify Address — Check first 6 and last 6 chars match destination
  • 2. Verify Network — ERC-20 vs BEP-20 vs BTC = lost forever if wrong
  • 3. Verify Amount — Send $10 test first, then full amount
  • 4. Verify Fee — Don't overpay gas during congestion
  • 5. Verify Device — Confirming on hardware wallet screen, not computer
Test Transaction Protocol: Send $10 → Wait for blockchain confirmation → Then send full amount. Reduces catastrophic error risk by 99%.

🎣 Layer 6: Anti-Phishing Fortress

No exchange has official WhatsApp support.
No "CEO" or "founder" DMs you on Telegram.
No customer support reaches out first.
  • Bookmark exchange URLs. Never click links in emails.
  • Verify anti-phishing code in every exchange email
  • Verify sender domain: @crypto.com ✓ | @crypto-support.com ✗
  • No exchange EVER asks for your password or seed phrase via email

📱 Layer 7: SIM Swap Protection

SMS 2FA is DEATH. Attacker ports your number → receives all SMS codes → drains account in minutes.
Replace SMS 2FA with: Google Authenticator (not Authy — centralized) or hardware security key (YubiKey)
  • Call Etisalat/du — request SIM port freeze
  • Require in-person ID verification for any number changes
  • Set account PIN (not birthday or 1234)
